# Customer-Data-Analysis


An end-to-end Customer Data Analysis project leveraging SQL and Power BI to analyze real-world shopping transaction data, identify customer behavior patterns, key revenue drivers, and generate actionable business insights.


Customer Data Analysis-Capstone project

This repository presents our first Data Analytics Capstone Project, developed during the Data Analytics Internship at DataMites, where real-world customer transaction data is analyzed using SQL and Power BI to generate actionable business insights.


Project Overview


This project is an end-to-end Customer Data Analysis designed to analyze real-world transactional data and uncover meaningful insights related to customer behavior, revenue generation, and purchasing patterns across multiple dimensions such as gender, age, product category, and payment methods.  
The analysis focuses on helping businesses make data-driven decisions using SQL and Power BI.
